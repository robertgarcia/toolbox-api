## How to run this project locally


### Required tools

- NodeJS runtime 14.21.3

### Optional tools
- Docker ^20.10.16
- Docker compose ^V2.5.0

### Install dependencies
- `npm install `

### Run with npm
- `npm run start`

### Run with docker
- `npm run start:docker`

## Access
- `http://localhost:8888`

### EndPoint
`/files/data`
